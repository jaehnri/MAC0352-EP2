# Tic Tac Toe - MAC0352-EP2
Tic tac toe game over the internet.

Second project for MAC0352 - Computer Networks and Distributed Systems.

## Build and Run
You need golang installed to run this program.

**Run Client:**
```sh
make build-server
make run-server
```

**Run Server:**
```sh
make build-client
make run-client
```

## Test
```sh
make test
```
